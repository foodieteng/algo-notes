01

題目簡介// Brief

QUESTION

給定正整數 N，輸出一個 1..N 的排列 a[1..N]，使得 不存在任意三個下標 i < j < k 滿足 2·a[j] = a[i] + a[k]（即 a[i], a[j], a[k] 在原順序下構成等差數列，簡稱 3-AP）。 存在多種合法答案時輸出任一即可。

用 n = 3 的所有排列做暖身（共 6 種）— 只有「全升」與「全降」是 3-AP，其他 4 種都合法：

排列 是否 3-AP? 收錄?
(1, 2, 3)是 — (1,2,3) 等差 d = +1×
(1, 3, 2)無 ●
(2, 1, 3)無 ●
(2, 3, 1)無 ●
(3, 1, 2)無 ●
(3, 2, 1)是 — (3,2,1) 等差 d = −1×

03

核心策略 · 奇偶切// Odd-Even Split

把 1..N 按奇偶分兩組：所有奇數放前段、所有偶數放後段。 每段內部再遞迴解一次相同問題即可。

  1. 切半 · half = ⌈N/2⌉，遞迴解 f(half) 與 f(N − half)
  2. 映射 · 前段 x → 2x − 1（變奇數）、後段 x → 2x（變偶數）
  3. 串接 · f(N) = [前段奇數] ∥ [後段偶數]

Base case · f(1) = [1]，輸入縮到單一元素時直接收尾。

06

為什麼奇偶切保證 3-AP free// Proof Sketch

假設存在逆例 (i, j, k) 滿足 2·a[j] = a[i] + a[k]。 右式必為偶數 ⇒ a[i] 與 a[k] 同奇偶。分以下三種情況討論：

// PROOF · 奇偶切保證 3-AP free
  1. Case ① 兩端皆奇 ⇒ 都在前段，j 介於兩者亦在前段 ⇒ a[j] 也是奇數。整組 3-AP 都落在前段 — 規模為 ⌈N/2⌉，由歸納假設不存在。
  2. Case ② 兩端皆偶 ⇒ 都在後段，同理 j 在後段 ⇒ a[j] 也是偶數。整組落在後段 — 規模為 ⌊N/2⌋，由歸納假設不存在。
  3. Case ③ 一奇一偶 ⇒ 違反「同奇偶」的必要條件 — 不可能發生。

三種情況都導致矛盾 ⇒ 逆例不存在 ⇒ 構造出的序列 3-AP free。 ▪

關鍵奇偶律：奇+奇=偶 · 奇+偶=奇 · 偶+偶=偶。 3-AP 中項是兩端平均 ⇒ 兩端必同奇偶 ⇒ 跨段不可能。

07

遞迴結構 · 公式// Recurrence

把上一段策略寫成精確的串接公式：

f(N) = [ 2 · f(⌈N/2⌉) − 1 ]  ∥  [ 2 · f(⌊N/2⌋) ]
∥ 為串接；2x − 1 把長 ⌈N/2⌉ 的排列映成奇數段；2x 映成偶數段

時間遞迴式：

T(N) = T(⌈N/2⌉) + T(⌊N/2⌋) + O(N) = O(N log N)
Master Theorem case 2 — 每層合併工作量 O(N)，共 log N 層
項目數量備註
TimeO(N log N)每層串接 O(N)，共 log N 層
SpaceO(N)最終輸出陣列；遞迴堆疊 O(log N)
Recursion DepthO(log N)N = 10⁶ ⇒ 約 20 層

08

一句洞察// One-Sentence Insight

奇偶切的真正力量 · 3-AP 的中項是兩端平均，而平均要落在整數空間， 兩端必同奇偶。把奇偶物理分離後，跨段 3-AP 立刻違反「中項位置在兩端之間」的位置條件 — 跨段零個 3-AP。只剩同段內部 — 而那是規模一半的同一個問題，遞迴解掉。
