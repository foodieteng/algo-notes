完整實作// Full C++

互動式題目：實作 solve(N, X, Y)，由 judge 提供 Report(...)。不需 main()。

p114.cpp C++17 
#include <iostream>
#include "lib0114.h"
using namespace std;

// 鋪滿左上角為 (top, left)、邊長 n 的子板；
// (blockedX, blockedY) 是已被佔用、不能再放 L 的格子。
void tile(int top, int left, int n, int blockedX, int blockedY) {
    if (n == 2) {
        // 4 格扣掉 blocked 後剩 3 格本身就是 L
        int xs[3], ys[3], k = 0;
        for (int dx = 0; dx < 2; ++dx) {
            for (int dy = 0; dy < 2; ++dy) {
                int x = top + dx, y = left + dy;
                if (x == blockedX && y == blockedY) continue;
                xs[k] = x; ys[k] = y; ++k;
            }
        }
        Report(xs[0], ys[0], xs[1], ys[1], xs[2], ys[2]);
        return;
    }

    const int half = n / 2;
    const int midX = top  + half - 1;   // 上半最後一列
    const int midY = left + half - 1;   // 左半最後一行

    // 4 個象限「最靠近中心」的格子 (TL, TR, BL, BR)
    int cx[4] = { midX,     midX,     midX + 1, midX + 1 };
    int cy[4] = { midY,     midY + 1, midY,     midY + 1 };

    // blocked 在哪個象限？  0 = TL, 1 = TR, 2 = BL, 3 = BR
    const int bq = (blockedX > midX) * 2 + (blockedY > midY);

    // 中央 L：不含 blocked 的 3 個象限的最角落
    int lx[3], ly[3], k = 0;
    for (int q = 0; q < 4; ++q) {
        if (q == bq) continue;
        lx[k] = cx[q]; ly[k] = cy[q]; ++k;
    }
    Report(lx[0], ly[0], lx[1], ly[1], lx[2], ly[2]);

    // 對 4 個象限分別遞迴；blocked 不在的象限以「剛種的角」當新 blocked
    int ox[4] = { top,  top,         top + half, top + half };
    int oy[4] = { left, left + half, left,       left + half };
    for (int q = 0; q < 4; ++q) {
        int nx = (q == bq) ? blockedX : cx[q];
        int ny = (q == bq) ? blockedY : cy[q];
        tile(ox[q], oy[q], half, nx, ny);
    }
}

void solve(int N, int X, int Y) {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    tile(1, 1, N, X, Y);
}

複雜度分析// Complexity Derivation

每呼叫一次 tile() 的工作量（不含遞迴）：

步驟內容單呼叫成本
base case 判斷n == 2 時掃 4 格O(1)
算 midX / midY / bq4 個位元運算 + 比較O(1)
列舉 4 角 + Report固定 4 次迴圈，1 次 ReportO(1)
遞迴 4 子板4 個 n/2 大小的子問題4 · T(N/2)
T(N) = 4 · T(N/2) + O(1)
Master Theorem case 1：a = 4, b = 2, log_b a = 2；f(N) = O(N⁰) < N² ⇒ T(N) = Θ(N²)
項目數量備註
TimeΘ(N²)總 (N² − 1)/3 個 L，每個 O(1)。
SpaceO(log N)只用遞迴堆疊；無額外陣列。
Recursion Depthlog₂ N = KN = 1024 ⇒ 10 層，stack 很淺。
Constant factor低純整數比較與算術；無 STL 容器。

下限：要放 (N² − 1)/3 = Θ(N²) 個 L，每個必須 ≥ O(1) ⇒ 演算法已達理論下限。

03

邊界與陷阱// Pitfalls

陷阱說明 / 解法
1-indexed 座標 題目給的 (X, Y) 從 1 開始，最外層呼叫必須是 tile(1, 1, N, X, Y)，不是 (0, 0, ...)。
midX 是上半最後一列 midX = top + half − 1，不是中線。下半從 midX + 1 開始；4 個角是 {midX, midX+1} × {midY, midY+1}。
bq 編號用比較 bq = (blockedX > midX) * 2 + (blockedY > midY) — 高位是 X、低位是 Y，剛好 TL=0, TR=1, BL=2, BR=3。注意是 > midX（嚴格大於）。
base case 不需算 bq n = 2 時不切象限；雙重 for 跳過 blocked 即可。提早判斷 n == 2 是必須的，否則 half = 1、midX = top、會繼續往下遞迴變成無限迴圈。
中央 L 必須先 Report 放完中央 L 才能讓其他 3 個象限「有 blocked」可遞迴；順序顛倒會導致子象限沒有 blocked 而行為未定義。
不用 std::array 用原始 int cx[4] 等平行陣列，讓索引 0..3 直接對應 TL..BR — 比 struct 清楚。
不需 cin / cout 互動式 — 全靠 Report，但 ios::sync_with_stdio(false); cin.tie(nullptr); 仍習慣加上（無害）。

細節走讀// Section Walkthrough

① base case (n == 2)

// n == 2
double for-loop 掃 4 格 (top..top+1) × (left..left+1)
跳過 blocked，把剩 3 格塞進 xs / ys
一次 Report(xs[0], ys[0], xs[1], ys[1], xs[2], ys[2])
return — stack 在這停。

② 算 midX / midY (子板的中心邊界)

// split
half = n / 2
midX = top  + half - 1   ← 上半最後一列
midY = left + half - 1   ← 左半最後一行

4 個「最靠近中心」的角 (TL, TR, BL, BR)：
  cx = { midX,   midX,   midX+1, midX+1 }
  cy = { midY,   midY+1, midY,   midY+1 }

③ 判定 blocked 在哪個象限

// bq
bq = (blockedX > midX) * 2 + (blockedY > midY)

  TL = 0 :  blockedX ≤ midX  且  blockedY ≤ midY
  TR = 1 :  blockedX ≤ midX  且  blockedY > midY
  BL = 2 :  blockedX > midX  且  blockedY ≤ midY
  BR = 3 :  blockedX > midX  且  blockedY > midY

④ 放中央 L

// place center L
for q in 0..3:
    if q == bq: skip
    else: 把 (cx[q], cy[q]) 收進 lx / ly

Report(lx[0], ly[0], lx[1], ly[1], lx[2], ly[2])

→ 這 3 格從此被「占用」 ─ 在各自象限的遞迴裡會當新 blocked。

⑤ 遞迴 4 個子象限

// recurse 4 quadrants
ox = { top,  top,       top+half, top+half }   ← 子板左上 X
oy = { left, left+half, left,     left+half }  ← 子板左上 Y

for q in 0..3:
    if q == bq:
        nx, ny = blockedX, blockedY      ← 用原 blocked
    else:
        nx, ny = cx[q], cy[q]            ← 用剛種的角
    tile(ox[q], oy[q], half, nx, ny)

→ 每個子板都是「n/2 × n/2 + 1 個 blocked」 ─ 與原問題同形。
