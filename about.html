CLOUDFLARE PAGES + ACCESS
  • 將 repo 設為 GitHub Private。
  • 到 Cloudflare Pages 連結 repo，build 設定留空，輸出目錄填 /。
  • Pages 部署完成後到 Zero Trust → Access → Applications 建立 Self-hosted Application，把 Pages 網域加進去，設定 Email Policy 只允許自己的信箱。
Cloudflare Access 免費方案支援 50 個使用者，個人用綽綽有餘。
