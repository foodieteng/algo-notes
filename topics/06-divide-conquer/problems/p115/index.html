P115 MERGE SORT 2026.05.25
Divide & Conquer Merge Sort Inversion Pairs Modular O(N log N)

題意拆解

給一個長度為 N 的正整數序列 a[1..N]。 定義「逆序對」為一組下標 (i, j)， 其中 i < j 但 a[i] > a[j]。

請對每一個逆序對，計算它的兩個值 a[i] + a[j]， 然後把所有逆序對的這個和加起來，最後輸出 mod 10000019 之餘數。

範例：序列 (5, 1, 4, 2, 3) 中，(4, 3) 是一組逆序對 （4 出現在 3 之前，但 4 > 3），這組的和為 4 + 3 = 7。 加總所有這樣的和即為答案。

輸入 / 輸出

標準輸入兩行：

  • 第 1 行：正整數 N。
  • 第 2 行：N 個以空白分隔的正整數，每個 ≤ 10⁹。

輸出一個整數：所有逆序對之兩數和的總和，mod 10000019。

限制

  • 1 ≤ N ≤ 1,000,000
  • 每個 a[i] ≤ 10⁹
  • 30% 資料：1 ≤ N ≤ 1,000 — 可用 O(N²) 暴力
  • 70% 資料：1 ≤ N ≤ 45,678 — 需 O(N log N)
  • 100% 資料：N 最大 10⁶ — 需 O(N log N) + 注意 mod 不溢位

範例

// INPUT
5 5 4 2 1 3
// OUTPUT
51

範例驗算：序列 [5, 4, 2, 1, 3] 的所有逆序對及其兩數和：

  • (5,4) → 9 ／ (5,2) → 7 ／ (5,1) → 6 ／ (5,3) → 8
  • (4,2) → 6 ／ (4,1) → 5 ／ (4,3) → 7
  • (2,1) → 3
  • 合計：9 + 7 + 6 + 8 + 6 + 5 + 7 + 3 = 51
