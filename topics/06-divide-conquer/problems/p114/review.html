[ 5.1 ] Concept Recap

重要概念回顧

關鍵點內容
數學基礎 4 ≡ 1 (mod 3) ⇒ 4ᴷ − 1 ≡ 0 (mod 3)，L 數量 (N² − 1) / 3 必為整數。
分治架構 切 4 個 N/2 × N/2 象限；blocked 必落在恰好 1 個象限裡。
中央 L 的作用 在「不含 blocked」的 3 個象限角落各種一格 — 變成各象限的「新 blocked」，子問題完全同形。
Base case n = 2 時，4 格扣 blocked 剩 3 格本身就是 L，直接 Report。
時間複雜度 T(N) = 4·T(N/2) + O(1) ⇒ Θ(N²)（Master Theorem case 1）。
空間複雜度 遞迴深度 O(log N) = K；N=1024 只有 10 層。

// ANSWERS

先自己想過再對答案。下面按題號編排。

A1 · 數學

4 ≡ 1 (mod 3)，所以 4ᴷ ≡ 1ᴷ ≡ 1 (mod 3)， 因此 4ᴷ − 1 ≡ 0 (mod 3)，即 (N² − 1) / 3 必為整數。

A2 · 分治

中央 L 的 3 個 cell 是「不含 blocked」的 3 個象限各自的「最靠近中心」角落。 程式用 bq 把 blocked 所在象限編號出來，再用 if (q == bq) continue; 跳過它。 因此 blocked 那一象限不會有中央 L 的 cell。

A3 · 構造

放完中央 L 後：

  • 含原 blocked 的象限：仍只有 1 個 blocked（原本的 (X, Y)）。
  • 其他 3 個象限：被中央 L 在最角落種了 1 個 cell — 那一格就成為該象限的「新 blocked」。

每個子象限都是「N/2 × N/2 板 + 恰好 1 個 blocked」，跟原問題同形。 因此可以放心遞迴。

A4 · Base case

2×2 中任意 3 個格子不可能共線，因為：

  • 同一列只有 2 格（c 從 1 到 2）。
  • 同一行只有 2 格。

因此扣掉 1 格剩 3 格時，這 3 格必然包含至少一個轉折 — 就是 L 型。

A5 · 複雜度

Master Theorem 形式：T(N) = a·T(N/b) + f(N)，這裡 a = 4, b = 2, f(N) = O(1) = O(N⁰)。

計算 log_b a = log₂ 4 = 2。 因為 f(N) = O(N⁰) 且 0 < 2，進入 case 1 ⇒ T(N) = Θ(N^{log_b a}) = Θ(N²)。

直覺：總共要放 (N² − 1)/3 = Θ(N²) 個 L，每個 L 是 O(1) 工作，所以總工時 Θ(N²)。

A · Ex 1

solve(2, 2, 2) 直接命中 base case：4 格 (1,1), (1,2), (2,1), (2,2) 扣掉 blocked (2,2) 剩 3 格。 

Report(1, 1,
       1, 2,
       2, 1);

三格的順序可能依 double-loop 順序略異，但只要是這 3 格組合就正確。

A · Ex 2

逐步算 solve(4, 1, 1)：

tile(1, 1, 4, 1, 1)
  half = 2, midX = 2, midY = 2
  bq = (1 > 2) * 2 + (1 > 2) = 0 * 2 + 0 = 0    → TL

  中央 L = q ∈ {1, 2, 3} 的角落：
    q=1 (TR): (midX,   midY+1) = (2, 3)
    q=2 (BL): (midX+1, midY  ) = (3, 2)
    q=3 (BR): (midX+1, midY+1) = (3, 3)
  ▸ Report(2, 3,  3, 2,  3, 3)

遞迴 4 象限：
  TL (q=0)  tile(1, 1, 2, 1, 1)   → 原 blocked
  TR (q=1)  tile(1, 3, 2, 2, 3)   → 用剛種的
  BL (q=2)  tile(3, 1, 2, 3, 2)   → 用剛種的
  BR (q=3)  tile(3, 3, 2, 3, 3)   → 用剛種的

每個都命中 base case：

  ▸ tile(1, 1, 2, 1, 1)  →  Report(1, 2,  2, 1,  2, 2)
  ▸ tile(1, 3, 2, 2, 3)  →  Report(1, 3,  1, 4,  2, 4)
  ▸ tile(3, 1, 2, 3, 2)  →  Report(3, 1,  4, 1,  4, 2)
  ▸ tile(3, 3, 2, 3, 3)  →  Report(3, 4,  4, 3,  4, 4)

共 5 個 Report — 對應 (4² − 1) / 3 = 5。
A · Ex 3

solve(4, 2, 3) 第 1 層 trace：

tile(1, 1, 4, 2, 3)
  half = 2, midX = 2, midY = 2
  bq = (2 > 2) * 2 + (3 > 2) = 0 * 2 + 1 = 1   → TR

  中央 L = q ∈ {0, 2, 3} 的角落：
    q=0 (TL): (midX,   midY  ) = (2, 2)
    q=2 (BL): (midX+1, midY  ) = (3, 2)
    q=3 (BR): (midX+1, midY+1) = (3, 3)

▸ Report(2, 2,  3, 2,  3, 3)
A · App 1

套公式 (N² − 1) / 3：

  • N = 8    ⇒ (64 − 1) / 3 = 21 個 L
  • N = 16  ⇒ (256 − 1) / 3 = 85 個 L
  • N = 1024 ⇒ (1,048,576 − 1) / 3 = 349,525 個 L
A · App 2

遞迴深度 = log₂ N。 N = 1024 = 2¹⁰，所以最深 10 層（從 1024 → 512 → 256 → ... → 4 → 2）。

A · App 3

N = 8, half = 4, midX = 4, midY = 4：

bq = (5 > 4) * 2 + (7 > 4)
   = 1 * 2 + 1
   = 3                  → BR

blocked (5, 7) 位於 右下 (BR) 象限。
