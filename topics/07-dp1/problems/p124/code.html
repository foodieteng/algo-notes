完整實作 · 單一遞迴預處理// Full C++

最省事的寫法：直接用消去後的 f[n] = 4·f[n−2] − f[n−4]，一次填好 f[1..maxN]。

p124.cpp C++17 
#include <bits/stdc++.h>
using namespace std;

const int MOD = 1000007;
const int MAXN = 100005;

long long f[MAXN];   // f[n] = 用 1×2 骨牌鋪滿 3×n 的方法數 (mod)

void build() {
    f[0] = 1;                          // 空走廊：1 種（什麼都不放）
    if (MAXN > 1) f[1] = 0;            // 3×1 = 3 格，奇數鋪不滿
    if (MAXN > 2) f[2] = 3;            // 3×2 = 3 種（base case）
    if (MAXN > 3) f[3] = 0;            // 奇數 → 0

    for (int n = 4; n < MAXN; n++) {
        // 減法後可能變負 ⇒ + MOD 再取餘
        f[n] = ((4 * f[n-2] - f[n-4]) % MOD + MOD) % MOD;
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    build();                          // 一次預處理所有 N

    int T;
    cin >> T;
    while (T--) {
        int n;
        cin >> n;
        cout << f[n] << '\n';     // 查表 O(1)
    }

    return 0;
}

理解版（耦合兩狀態）：若想直接照「平 / 缺一角」兩種邊界寫，用 f、g 兩條陣列同步遞推：

p124_fg.cpp C++17 
long long f[MAXN], g[MAXN];
f[0] = 1; f[1] = 0;
g[0] = 0; g[1] = 1;
for (int n = 2; n < MAXN; n++) {
    f[n] = (f[n-2] + 2 * g[n-1]) % MOD;
    g[n] = (f[n-1] + g[n-2]) % MOD;
}
// 答案一樣是 f[N]，且沒有減法、不必補 MOD

03

邊界與陷阱// Pitfalls

陷阱說明 / 解法
奇數 N → 0 3×N 共 3N 格，N 奇數 ⇒ 格數奇數、骨牌蓋偶數格 ⇒ 鋪不滿。遞迴本身會自動算出 0，但要確認 base 正確。
單一遞迴的減法會變負 4f[n−2] − f[n−4] 取餘後可能 < 0 ⇒ (x % MOD + MOD) % MOD。（用 f/g 雙狀態版則無減法、不必補。）
模數是 1000007 對 1000007 取餘，不是 1e9+7。它非質數（=29×34483），但只有加減乘、用不到逆元。
f[0] = 1 空走廊算「1 種鋪法」是遞迴正確的基底；設成 0 會讓整條遞迴歪掉。
多筆要預處理 T 筆各自 O(N) 會 TLE；一次 build 全部 N、查表回答。
10⁵ 筆 I/O sync_with_stdio(false); cin.tie(nullptr);，輸出用 '\n'。
04

細節走讀// Section Walkthrough

① 兩種邊界狀態

// states
f[n] = 3×n 完整鋪滿（右邊界是平的）的方法數
g[n] = 前 n-1 欄全滿、最後一欄缺頂端一格（缺一角／凹）的方法數
答案 = f[N]

② 耦合遞推

// recurrence
f[n] = f[n-2] + 2·g[n-1]
       // 最左欄三橫填滿 2 欄(→f[n-2])；或留下缺角邊界(缺頂/缺底 上下對稱 ×2)
g[n] = f[n-1] + g[n-2]
       // 缺角正下方那格：直放→f[n-1]；橫放→又冒出缺角 g[n-2]

③ 消去 g 得單一遞迴

// eliminate g
由兩式代入消去 g，可得：
f[n] = 4·f[n-2] − f[n-4]
// 只用 f，一條陣列就好（代價：出現減法要補 MOD）

④ 為什麼奇數一定 0

// parity
總格數 = 3 × N
每塊骨牌蓋 2 格 ⇒ 鋪滿需要總格數為偶
3N 偶 ⇔ N 偶 ⇒ N 奇 ⇒ f[N] = 0
（遞迴 base f[1]=f[3]=0 會把奇數鏈全壓成 0）
