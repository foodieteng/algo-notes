[ 01 ] Problem

題意拆解

給一個長度為 n 的整數陣列 a[0..n-1]（元素可為負）。找一段連續區間 [l, r]，使得 a[l] + a[l+1] + ... + a[r] 最大，並輸出此最大值。

輸入 / 輸出

// INPUT
9 -2 1 -3 4 -1 2 1 -5 4
// OUTPUT
6

限制

  • 1 ≤ n ≤ 10⁵
  • −10⁴ ≤ a[i] ≤ 10⁴
  • 必須選至少一個元素（不可空區間）
最佳子陣列為 a[3..6] = [4, -1, 2, 1]，總和為 6。
[ 02 ] Concept

演算法概念

直覺發想

暴力做法是枚舉所有區間 (l, r) 共 O(n²) 對，逐一求和 O(n)，總共 O(n³)。利用前綴和可優化到 O(n²)。但我們真正在乎的，其實是「以 i 為結尾的最佳區間和」。

定義狀態：

dp[i] = 以 a[i] 結尾的最大子陣列和

狀態轉移：

dp[i] = max( a[i], dp[i-1] + a[i] )

答案即為 max(dp[0..n-1])。

為什麼這樣對？

對任意以 i 結尾的最佳區間 [l, i]：

  • 若 l == i，則 dp[i] = a[i]
  • 若 l < i，則 [l, i-1] 必然是以 i-1 結尾的最佳區間，否則可以替換得到更好的答案 — 矛盾。

因此 dp[i] 只需比較 dp[i-1] + a[i] 與 a[i]。

複雜度

Time O(n) Space O(1)

只需單一變數 cur 滾動 dp[i-1]，故空間從 O(n) 壓縮到 O(1)。

[ 04 ] Clean Code

Clean Code

kadane.cpp C++ 
#include <bits/stdc++.h>
using namespace std;

int max_subarray_sum(const vector<int>& a) {
    int cur = a[0];
    int best = a[0];
    for (int i = 1; i < (int)a.size(); ++i) {
        cur = max(a[i], cur + a[i]);
        best = max(best, cur);
    }
    return best;
}

int main() {
    int n;  cin >> n;
    vector<int> a(n);
    for (int& x : a) cin >> x;
    cout << max_subarray_sum(a) << '\n';
    return 0;
}
kadane.py Python 
def max_subarray_sum(a: list[int]) -> int:
    cur = best = a[0]
    for x in a[1:]:
        cur = max(x, cur + x)
        best = max(best, cur)
    return best


if __name__ == "__main__":

[ 05 ] Walkthrough

Code 走讀

逐段拆解 kadane.cpp：

int cur = a[0]; int best = a[0];
初始化兩個變數：
· cur = 以當前位置為結尾的最大區間和
· best = 目前看過的全域最大
都以 a[0] 起始 — 題目保證至少要選一個元素，故不能用 0。
for (int i = 1; i < (int)a.size(); ++i) {
從 i=1 開始掃，因為 i=0 已在初始化處理。
cur = max(a[i], cur + a[i]);
狀態轉移：要不要「延續」前面的區間。
cur + a[i] < a[i] ⇔ cur < 0，前面是負擔就拋棄重起。
best = max(best, cur); }
更新全域最佳。注意是 每一步都更新，而非結尾才更新。
// Common Pitfalls
  • best 初始化用 0 在「全為負數」時會錯誤回傳 0。
  • 若改成允許空區間，需顯式判斷或將 best 初始化為 0。
  • 累加會溢位時用 long long（本題 n×|a[i]| ≈ 10⁹，剛好邊界）。
[ 06 ] Variants

延伸 / 變形題

變形差異解法概念
輸出區間端點 [l, r] 不只求和，要回傳起訖 追蹤 curL；當 cur 重起時更新 curL = i；best 更新時記下 (curL, i)
環形陣列 區間可跨越尾端接到開頭 答案 = max(直線版 Kadane, 總和 − 最小子陣列和)；特例：全為負
二維最大子矩陣 從一維推廣到 m×n 矩陣 固定上下界，把列壓成一維後跑 Kadane；O(m²·n)
長度限制 k 區間長度 ≤ k 前綴和 + 單調隊列維護視窗最小前綴
