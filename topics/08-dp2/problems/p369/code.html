01

完整實作 · 距離狀態 + 後綴和// Full C++

輸入一行 N S B K。狀態不是「樓層」而是「到禁止層 b 的距離 a = |x−b|」（因為永遠跨不過 b，只有 S 那一側有關）。 f[a] 是停在距離 a 的方法數。每趟轉移 nf[a'] = (Σ_{a ≥ amin} f[a]) − f[a']，用後綴和 O(L) 完成整趟。

p369.cpp C++17 
#include <bits/stdc++.h>
using namespace std;
const long long MOD = 1000000007;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    long long N, S, B, K;
    cin >> N >> S >> B >> K;        // 輸入順序：N S B K

    if (S == B) { cout << 0 << '\n'; return 0; }   // 起點就是禁止層

    // 每趟都跨不過 B，所以只跟 S 那一側有關。
    // 狀態 = 到 B 的距離 a ∈ [1, L]。
    long long L, a0;
    if (S < B) { L = B - 1; a0 = B - S; }   // 下側 floors 1..B-1
    else       { L = N - B; a0 = S - B; }   // 上側 floors B+1..N

    if (K == 0) { cout << 1 << '\n'; return 0; }    // 沒跳就只有 1 種

    // 滾動陣列：f 是目前這排，nf 是下一排，suf 是後綴和
    vector<long long> f(L + 2, 0), nf(L + 2, 0), suf(L + 2, 0);
    f[a0] = 1;                                       // f_0：起點在距離 a0

    for (long long step = 0; step < K; ++step) {
        // (1) 後綴和 suf[j] = Σ_{a>=j} f[a]
        suf[L + 1] = 0;
        for (long long j = L; j >= 1; --j)
            suf[j] = (suf[j + 1] + f[j]) % MOD;

        // (2) 轉移 f_{t+1}(a') = suf[門檻] - f(a')
        for (long long b = 1; b <= L; ++b) {
            long long amin = (b + 2) / 2;            // = ceil((b+1)/2)
            nf[b] = ((suf[amin] - f[b]) % MOD + MOD) % MOD;
        }
        swap(f, nf);                                 // 滾動到下一步
    }

    long long ans = 0;
    for (long long b = 1; b <= L; ++b) ans = (ans + f[b]) % MOD;
    cout << ans << '\n';
    return 0;
}

為什麼門檻是 amin = ceil((a'+1)/2)？ 從距離 a 跳到距離 a'，合法條件 |y−x| < |b−x| 化簡成 a' < 2a（同側、嚴格小於）。反解出來源 a：a > a'/2 ⟺ a ≥ ⌈(a'+1)/2⌉。所以 a' 接收的是 所有距離 ≥ amin 的 f[a]，正好是一段後綴。最後扣 f[a'] 是因為電梯要動（a≠a'）。

02

複雜度分析// Complexity Derivation

步驟內容成本
每趟後綴和從 L 掃到 1 累加 suf[]O(L)
每趟轉移每個 a' 一次 suf[amin]−f[a']O(L)
共 K 趟—O(L·K)
最後加總Σ f[a]O(L)
K 趟 × O(L)/趟 = O(L·K) ≤ O(N·K)
L < N（只取一側）；2000×2000 = 4×10⁶，遠快於樸素 O(N²·K)=1.6×10¹⁰
項目數量備註
TimeO(L·K)後綴和把整趟轉移降到 O(L)
SpaceO(L)f / nf / suf 三條滾動陣列
遞迴深度0（迭代）逐趟前進，swap 滾動
Constant factor低每趟兩次線性掃
03

邊界與陷阱// Pitfalls

陷阱說明 / 解法
輸入順序 N S B K 這版用 N S B K（起點 S 在禁止層 B 之前）。別跟其它版本的 n a b k 搞混。
只取一側 S<B ⇒ 下側 L=B−1, a0=B−S；S>B ⇒ 上側 L=N−B, a0=S−B。另一側永遠到不了，根本不用建。
門檻取整 amin=(b+2)/2 是整數除法，恰等於 ⌈(b+1)/2⌉。寫成 (b+1)/2 會少一格、把不合法來源算進去。
別忘扣 f[a'] 後綴 suf[amin] 含 a'=a 自己（距離不變的「停原地」），但電梯要動 ⇒ nf[a']=suf[amin]−f[a']。
減法補 MOD suf[amin]−f[a'] 可能為負 ⇒ ((… )%MOD+MOD)%MOD。
K=0 / S=B 特判 K=0：沒跳，答案 1。S=B：起點就在禁止層，答案 0。兩者都先擋掉。
答案是 Σ f[a] 搭完 K 趟停在任一距離都算一種搭法，全部加起來。
04

細節走讀// Section Walkthrough

① 換軸：樓層 → 到 b 的距離

// reframe
條件 |y−x| < |b−x| ⇒ 每趟都跳不到也跳不過 b。
所以 S 在哪一側就永遠在那一側 → 另一側整段砍掉。
令 a = |x − b| ∈ [1, L]，f[a] = 停在距離 a 的方法數。
  S < B：L = B−1，   a0 = B − S
  S > B：L = N − B，  a0 = S − B

② 把條件化簡成 a' 的範圍

// derive threshold
同側兩點：|y−x| = |a − a'|，而 |b−x| = a。
合法 |a−a'| < a ⟺ 0 < a' < 2a ⟺ a' ∈ [1, 2a−1]。
反過來問「哪些來源 a 能到 a'」：a' ≤ 2a−1 ⟺ a ≥ ⌈(a'+1)/2⌉ = amin。
⇒ a' 收下「所有距離 ≥ amin 的 f[a]」= 一段後綴。

③ 一趟轉移（後綴和 O(L)）

// one ride
suf[j] = Σ_{a≥j} f[a]          // 後綴和，從 L 倒掃到 1
對每個目標 a' = 1..L：
  amin = (a'+2)/2
  nf[a'] = suf[amin] − f[a']    // 扣 a'=a 自身（電梯要動）
swap(f, nf)                     // 滾動到下一趟

④ 為什麼這比差分版更省

// vs diff-array
差分版：在樓層軸上，每個 x 把 cur[x] 灑到區間 [x−d+1,x+d−1]（O(L) 攤）。
距離版：a' 只是「收一段後綴」，連差分都不用，直接後綴和一次到位。
兩者都 O(L)/趟，但距離版陣列長度只有單側 L，常數更小、更好寫。
