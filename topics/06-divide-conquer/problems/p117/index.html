// Bad Array · 2×2 block 遞迴構造

★ ★ ★ ★ ☆ 核心 Divide & Conquer · Constructive · N = 2^K ≤ 1024 · 時間 Θ(N²)
PAGE 1 / 3  ·  CONCEPT
01

題目簡介// Brief

QUESTION

給 N = 2^K（N ≤ 1024），請構造一個 N × N 矩陣 A，每格填入 1..2N−1 之中的某個整數（可重複），且滿足： 對於每個 i，row i 與 col i 所有元素的聯集恰為 {1, 2, ..., 2N−1}。

互動式：實作 solve(N)，由 judge 提供 void Report(int)。 依 row-major 順序輸出 N² 個整數。

聯集大小是 2N−1；單一 row 或 col 有 N 格 — 所以 row 和 col 的元素之間必然大量重疊（剛好 1 個），這是構造能成立的容量條件。

02

B 隨機填 + 驗證 非確定性

C 2×2 block 遞迴構造 Θ(N²)

● ADOPT · Θ(N²)
03

核心策略 · 2×2 block// Algorithm Outline

設 K = N/2，shift = 2K − 1 = N − 1。把 A(N) 拆成 4 個 K × K 子塊：

子塊內容值域
TL (top-left)A(K)[i][j]{1..2K−1}
BR (bottom-right)A(K)[i][j]（同 TL）{1..2K−1}
BL (bottom-left)A(K)[i][j] + shift{shift+1..2N−2}
TR (top-right)i == j 填 2N − 1；否則 A(K)[i][j] + shift{shift+1..2N−1}
  1. 遞迴填 TL · build(K, 0, 0, base)
  2. 遞迴填 BR · build(K, K, K, base)（與 TL 同值，不平移）
  3. 掃 TL 推導 BL · M[K+i][j] = M[i][j] + shift
  4. 掃 TL 推導 TR · 對角線 (i == j) 填 2N − 1，其餘 M[i][K+j] = M[i][j] + shift

所有寫入都是 O(1)，每格只被填一次 ⇒ 總工作量 = N² ⇒ Θ(N²)。

06

構造正確性// Correctness

// CORRECTNESS · 為何 row i ∪ col i = {1..2N−1}

歸納假設：對 A(K)，每個 i ∈ [1..K] 滿足 row i ∪ col i = {1..2K−1}。

看 A(N) 的某一個 i ∈ [1..N]。分兩種情況：

  1. i ∈ [1..K]（上半 / 左半）
    • row i = TL[i] ∪ TR[i]。TL[i] 是 A(K) 的 row i。TR[i] 除了對角格 (i, K+i) = 2N−1，其餘 = A(K)[i] + shift（剝掉 j = i 那一格）。
    • col i = TL[*][i] ∪ BL[*][i]。TL[*][i] 是 A(K) 的 col i。BL[*][i] = A(K)[*][i] + shift（包含 j = i 那一格 = A(K)[i][i] + shift）。
    • 聯集：A(K) 的 row i ∪ col i = {1..2K−1}（歸納假設）。加上 +shift 的版本 = {shift+1 .. shift + 2K−1} = {2K .. 2N−2}。再加上對角的 2N−1。合起來 = {1..2K−1} ∪ {2K..2N−2} ∪ {2N−1} = {1..2N−1}。
    • 細節：TR[i] 缺了 j = i 那格的「A(K)[i][i] + shift」，但 BL[*][i] 補回來；對角線塞 2N−1 補上少的那個最大值。
  2. i ∈ [K+1..N]（下半 / 右半）
    • 令 i' = i − K ∈ [1..K]。row i = BL[i'] ∪ BR[i']。BR[i'] = A(K) 的 row i'。BL[i'] = A(K)[i'] + shift。
    • col i = TR[*][i'] ∪ BR[*][i']。TR[*][i'] 除了對角格 (i', K+i') = 2N−1，其餘 = A(K)[*][i'] + shift（缺 j = i' 那一格）。BR[*][i'] = A(K) 的 col i'。
    • 同樣聯集 = {1..2K−1} ∪ {2K..2N−2} ∪ {2N−1} = {1..2N−1} ✓。

所有值都落在 [1, 2N−1] ⇒ 構造合法。Q.E.D.

換句話說：shift = N − 1 把 A(K) 的值域 {1..2K−1} 平移到 {2K..2N−2}， 兩個值域剛好對接、合起來是 {1..2N−2}； TR 對角線塞 2N−1 是把最後那個最大值「灑」進每一個 row 和 col，補完最後一格。

07

複雜度// Complexity

每個 build(N) 呼叫：遞迴填 TL、BR 各一次（兩個 K × K），然後掃 TL 推導 BL、TR 共 2 · K² 次寫入。

T(N) = 2 · T(N/2) + Θ(N²/2)  ⇒  T(N) = Θ(N²)
每格只寫一次 ⇒ 總工作量 = 格數 = N²。Master Theorem case 3。
項目數量備註
TimeΘ(N²)每格 O(1) 寫入；N = 1024 ⇒ ~10⁶ 次運算
SpaceΘ(N²)全域 int M[1100][1100] ≈ 4.8 MB
Recursion DepthO(log N)N = 1024 ⇒ 10 層
Report CallsN²row-major 順序輸出

瓶頸是輸出 — N² 個 Report 呼叫無法再省。建構本身與輸出同階。

08

一句洞察// One-Sentence Insight

TR 對角線塞 2N−1 是把「最大值」灑進每個 row 和 col 的關鍵。
其他三塊（TL、BR、BL）合力撐起 {1..2N−2}， 對角的 2N−1 是唯一補上最後那一格的位置 — 每個 row i 都會碰到它一次，每個 col i 也是。
