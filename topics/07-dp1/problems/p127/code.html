01

完整實作 · 滾動兩變數// Full C++

不用開整條 dp[] — 只需「到上一格的最佳 take（取了上一格）/ skip（沒取上一格）」兩個值滾動更新。

p127.cpp C++17 
#include <bits/stdc++.h>
using namespace std;

void solve() {
    int n;
    cin >> n;

    // take = 「目前這格有取」的最佳和；skip = 「目前這格沒取」的最佳和
    long long take = 0, skip = 0;

    for (int i = 0; i < n; i++) {
        long long a;
        cin >> a;

        long long newTake = skip + a;          // 取這格 ⇒ 上一格必須沒取
        long long newSkip = max(take, skip);     // 不取這格 ⇒ 上一格取或不取都行，取大的

        take = newTake;
        skip = newSkip;
    }

    cout << max(take, skip) << '\n';   // 最後一格取或不取，取大的
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int T;
    cin >> T;
    while (T--) solve();

    return 0;
}

對應陣列版：dp[i] = max(dp[i−1], dp[i−2] + a[i])。這裡 skip 扮演 dp[i−1] 的「不取」分量、 take 扮演「取」分量；每步只往前滾一格，所以不必存整條陣列。

複雜度分析// Complexity Derivation

步驟內容成本

全部 T 筆—O(T · N)
每筆 O(N)，總 O(T · N)
T ≤ 10、N ≤ 10⁵ ⇒ 約 10⁶ 次運算，秒過
項目數量備註
TimeO(N) 每筆單遍掃描
SpaceO(1)邊讀邊算，連陣列都不用存

Constant factor低一次 max + 一次加法
03

邊界與陷阱// Pitfalls

陷阱說明 / 解法
不要用貪心 「每次拿最大、刪鄰居」會錯（[3,4,3,4,3] 貪心拿 4+4=8，但取兩端三個 3 = 9 更大）。一定要 DP。
skip 用 max(take, skip) 「不取這格」時，上一格可取可不取 ⇒ 要取兩者較大；不是只接 take。
take 接的是 skip 「取這格」⇒ 上一格必須沒取 ⇒ 接 skip + a，不是 max(take,skip)+a。
N = 1 只有一個數：第一輪後 take=a[0]、skip=0 ⇒ max = a[0]，正確。初值 take=skip=0 已涵蓋。
總和型別 aᵢ ≤ 1000、N ≤ 10⁵ ⇒ 和最大 10⁸，int 雖夠但用 long long 保險、習慣。
正數才能省略「少取」 本題 aᵢ ≥ 1 全正，所以「能取就取」不會虧。若有負數，base 與轉移要改（skip 也要能保留 0）。
04

細節走讀// Section Walkthrough

① 兩個狀態的意義

// state
take = 「處理到目前這格、且這格『有取』」的最大和
skip = 「處理到目前這格、且這格『沒取』」的最大和
答案 = max(take, skip)（最後一格取或不取）

② 轉移（讀到新的 a）

// transition
newTake = skip + a      // 取這格 → 上一格不能取 → 接 skip
newSkip = max(take,skip)// 不取這格 → 上一格隨意 → 取較大
take, skip = newTake, newSkip

注意要用舊的 take/skip 同時算出兩個新值，再一起更新 — 別先覆寫 take 再拿去算 skip。

③ 與 dp[] 陣列版的對應

// equivalence
dp[i] = max(dp[i-1], dp[i-2] + a[i])
      = max( 沿用上一格最佳 , 取這格 + 前前格最佳 )
滾動版把「dp[i-1] 的不取分量」存成 skip、
        「取了某格」的鏈存成 take，省掉整條陣列

ONE-LINE SUMMARY
滾動 take/skip：newTake=skip+a、newSkip=max(take,skip)，邊讀邊算 O(N)、空間 O(1)。
take 接 skip（取則鄰居不取）；skip 取 max（不取則鄰居隨意）。用 long long 保險。
