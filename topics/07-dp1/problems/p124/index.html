円円數磁磚

// 計數 DP · 3×N 骨牌鋪法

★ ★ ★ ☆ ☆ 核心 耦合狀態 DP（profile） · 時間 O(maxN + T)
PAGE 1 / 3  ·  概念
01

題目簡介// Brief

QUESTION

一條寬 3 塊磁磚、長 N 塊磁磚的走廊（即 3 × N 方格）， 要用 1×2 的長方形骨牌鋪滿，問共有幾種鋪法？答案對 1000007 取餘。 多筆 T ≤ 10⁵，1 ≤ N ≤ 10⁵。

範例：N=1 → 0、N=2 → 3、N=4 → 11、N=100000 → 232803。 關鍵觀察：3×N 共 3N 格，每塊骨牌蓋 2 格 ⇒ N 為奇數時 3N 是奇數、鋪不滿 ⇒ 0。

02

為什麼用 DP// Why DP

A 枚舉所有鋪法 指數級
逐格嘗試放骨牌再回溯 ⇒ 方法數本身就指數成長，N=100000 完全不可能列出。
× REJECT · TLE
B 每筆各自 DP O(T · N)
每筆測資重跑 O(N) ⇒ T·N 最壞 10⁵×10⁵ = 10¹⁰，TLE。
△ REJECT · TLE
BOTH FAIL · ADOPT
C 耦合狀態 DP + 預處理 O(maxN + T)
用「逐欄填、邊界只有兩種型態」推遞迴，一次算好 f[1..10⁵]，每筆查表 O(1)。
● ADOPT · STABLE

核心想法：一欄一欄往右鋪，欄與欄的交界只可能是「平的」或「凸出一格」少數幾種型態。 把這些型態當狀態，就能寫出線性遞迴。多筆 ⇒ 一次預處理所有 N。

03

狀態設計 · 兩種邊界// State Design

逐欄鋪，看「目前已鋪區域的右邊界」長什麼樣。3 寬的走廊，邊界本質上只有兩型：

  • f[n] = 把 3×n 完整鋪滿（平邊界）的方法數。
  • g[n] = 3×n 但多凸出一格（鋸齒邊界，少一格沒鋪）的方法數。
f[n] = f[n−2] + 2·g[n−1]
g[n] = f[n−1] + g[n−2]
base f[0]=1, f[1]=0, g[0]=0, g[1]=1；答案 = f[N]

（f[n]=f[n−2]+2g[n−1]：最後兩欄塞滿三橫或承接凸邊界； 兩式聯立消去 g 後得單一遞迴 f[n]=4f[n−2]−f[n−4]。）

05

遞迴怎麼來的// Deriving the Recurrence

先把 g 講清楚，再看 f[n]=f[n−2]+2·g[n−1] 的每一項長什麼樣。

STEP 1 g[k] 是什麼？「鋪好 k 欄、但多戳一格到第 k+1 欄」

g 之所以重要：鋪到一半時，「已鋪滿區域的右緣」要嘛是平的（f），要嘛是凸出一格（g）， 不會有別種形狀。所以只要追蹤這兩個量。

STEP 2 用範例拆解：f[2] = f[0] + 2·g[1] = 1 + 2 = 3

3×2 剛好有 3 種鋪法。把它們按「最右邊怎麼收尾」分成兩組 — 正好對應遞迴的 f[0] 一項和 2·g[1] 兩項：

所以 2·g[n−1] 的 2 就是「凸出的那一格在上、在下」兩種鏡像。 每一種凸邊界 g[n−1]，都恰好能被一塊骨牌補平成完整的 f[n] — 凸在上補一種、凸在下補一種，數量一樣多，故乘 2。

// 整理 · 兩條遞迴怎麼來的
  1. f[n] 的 (a) 項：最右邊用「三條橫」把整塊 3×2 收掉，前面剩一個平邊界 ⇒ 接 f[n−2]。
  2. f[n] 的 (b) 項：最右邊是一個凸邊界被一塊骨牌補平；凸在上 / 凸在下兩種鏡像 ⇒ 2·g[n−1]。
  3. 合起來 f[n] = f[n−2] + 2·g[n−1]。（n=2 驗證：1 + 2×1 = 3 ✓）
  4. g[n] 同理：一個平邊界 f[n−1] 再戳一塊出去、或承接更早的凸邊界 g[n−2] ⇒ g[n] = f[n−1] + g[n−2]。
  5. 兩式消去 g ⇒ f[n] = 4·f[n−2] − f[n−4]（實作最省事）。
f[n] = 4·f[n−2] − f[n−4]
base f[0]=1, f[2]=3；奇數 N → 0；對 1000007 取餘（減法後 +MOD）
06

複雜度// Complexity

預處理 O(maxN) + 每筆 O(1) = O(maxN + T)
maxN = 10⁵、T = 10⁵ ⇒ 約 2×10⁵ 步，秒過；對照每筆重算 O(T·N)=10¹⁰
項目數量備註
TimeO(maxN + T)預處理一次、查表 O(1)
SpaceO(maxN)f[] 一條陣列
遞迴深度0（迭代）純迴圈填表
Constant factor低整數加減取餘
07

一句洞察// One-Sentence Insight

逐欄鋪，交界只有「平」或「凸一格」兩種型態。
把型態當狀態 ⇒ f / g 互相遞推 ⇒ 消去後 f[n]=4f[n−2]−f[n−4]；
奇數 N 直接 0；多筆一次預處理查表。
