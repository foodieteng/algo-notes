玩電梯

// 計數 DP · 換軸成「到禁止層的距離」用後綴和轉移

★ ★ ★ ★ ☆ 核心 f[趟數][距離 a] + 後綴和 · 時間 O(L·K)
PAGE 1 / 3  ·  概念
01

題目簡介// Brief

QUESTION

N 層樓，你在樓層 S，樓層 B 是禁止進入的研究室。你想連續搭 K 趟超級電梯。 每趟從樓層 x 搭到 y，要求 |y − x| < |B − x|（搭乘距離小於你到 B 的距離 ⇒ 永遠跳不到也跳不過 B）。

第 i+1 趟的起點 = 第 i 趟的終點（連續鏈接）。兩種搭法視為相同 ⟺ 每一趟所在樓層都一樣。 問有幾種不同搭法，mod 1000000007。2 ≤ N ≤ 2000，S ≠ B，1 ≤ K ≤ 2000。

白話：一個「搭法」就是樓層序列 S = f₀, f₁, …, f_K，每步 |fᵢ − fᵢ₋₁| < |B − fᵢ₋₁|。 這條件保證每趟都跳不到 B（若 y=B 則 |y−x|=|B−x| 不 <），所以自動避開禁止樓層、且永遠停在 B 的同一側。求序列數。

02

為什麼換軸成「距離」+ 後綴和// Why Distance + Suffix Sum

A 枚舉所有樓層序列 O(Nᴷ)
每趟近 N 種落點，K 趟 ⇒ 指數爆炸；N,K=2000 完全不可能。
× REJECT · TLE
B DP 在樓層軸，逐對轉移 O(N²·K)
dp[趟][f] 對，但每步對每個 x 掃所有 y ⇒ N²·K ≈ 1.6×10¹⁰，TLE。
△ REJECT · TLE
BOTH FAIL · ADOPT
C 換軸成「到 B 的距離 a」+ 後綴和 O(L·K)
永遠跨不過 B ⇒ 只留 S 那一側，狀態改成距離 a=|x−B|。條件化簡成「a' 收下所有 a ≥ ⌈(a'+1)/2⌉ 的方法數」＝一段後綴，後綴和一次 O(L) 完成整趟。
● ADOPT · STABLE

兩個關鍵簡化：(1) 跨不過 B ⇒ 砍掉另一側，陣列長度只剩 L； (2) 用「距離」當軸後，每個目標 a' 收下的是 f[] 的一段後綴 {a ≥ amin}， 於是「對每個 a' 求一段區間和」就用後綴和 O(L)/趟，把樸素 O(N²) 直接壓掉。

03

核心策略 · 三步驟// Strategy

f[a] = 搭完目前這 s 趟後，停在「離 B 距離 a」的方法數（a∈[1,L]）。每趟做一次轉移：

  1. 換軸 · S<B ⇒ 下側 L=B−1, a0=B−S；S>B ⇒ 上側 L=N−B, a0=S−B。初值 f[a0]=1。
  2. 後綴和 · suf[j] = Σ_{a≥j} f[a]，從 a=L 倒掃到 1 累加。
  3. 轉移 + 扣自身 · 每個目標 a'：門檻 amin=⌈(a'+1)/2⌉，nf[a'] = suf[amin] − f[a']（扣掉停原地 a'=a）。
ft+1(a') = ( Σa ≥ ⌈(a'+1)/2⌉ ft(a) ) − ft(a')　（後綴和 O(L) 完成整趟）
重複 K 趟；答案 = Σ_a f[a]（停在任一距離都算一種搭法）。MOD = 1000000007（質數）。

05

為什麼門檻是 ⌈(a'+1)/2⌉// Why the Threshold

// 條件 |y−x| < |b−x| 在「同側距離軸」上的化簡
  1. x、y 同在 B 的一側，距離分別是 a=|x−B|、a'=|y−B|，於是 |y−x| = |a − a'|、|B−x| = a。
  2. 合法條件 |a − a'| < a ⟺ −a < a−a' < a ⟺ 0 < a' < 2a，即 a' ∈ [1, 2a−1]。
  3. 從「來源 a」看：a 能到達的目標是 [1, 2a−1]；反過來「目標 a' 收得到誰」：需 a' ≤ 2a−1 ⟺ a ≥ (a'+1)/2 ⟺ a ≥ ⌈(a'+1)/2⌉ = amin。
  4. 所以 a' 收下的來源是 {a : a ≥ amin}＝ f[] 的一段後綴。整數除法 amin=(a'+2)/2 恰好等於 ⌈(a'+1)/2⌉。
  5. 後綴含 a'=a 自己（停原地，a'∈[1,2a−1] 因 a≥1），但電梯要動 ⇒ 扣掉 f[a']。

「每個 a' 收下一段後綴」⇒ 先算一次後綴和 suf[]，每個 a' 只查 suf[amin] 是 O(1)，整趟 O(L)、K 趟 O(L·K)。 連差分都省了 — 因為後綴和正是「對每個位置查右側區間和」的工具。

06

複雜度// Complexity

K 趟 × 每趟 O(L)（後綴和 + 轉移）= O(L·K) ≤ O(N·K)
L < N（只取一側）；N,K ≤ 2000 ⇒ ≤ 4×10⁶，瞬殺；樸素 O(N²·K)=1.6×10¹⁰ 會 TLE
項目數量備註
TimeO(L·K)後綴和讓整趟轉移 O(L)
SpaceO(L)f / nf / suf 三條陣列滾動
遞迴深度0（迭代）逐趟前進，swap 滾動
Constant factor低每趟兩次線性掃，且只掃單側

MOD = 1000000007 是質數，但本題只用加減、沒有除法，所以不涉及逆元；減法記得 +MOD 防負。

07

一句洞察// One-Sentence Insight

跨不過 B ⇒ 換軸成「到 B 的距離 a」，把問題壓成單側一條陣列。
條件化簡後，每個目標 a' 收下的是 {a ≥ ⌈(a'+1)/2⌉} 的一段後綴 —
後綴和一次 O(L) 完成整趟、扣掉停原地 f[a']、答案是 Σ f[a]，總 O(L·K)。
