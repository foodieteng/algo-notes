Problem A: 王老先生

王老先生有一塊很大的正方形土地，邊長為 N，且 N 必為 2 的正整數次方 （即存在正整數 K 使 N = 2K）。

王老先生把整塊地切成 N×N 個邊長為 1 的方格， 位置以 (i, j) 表示，意思是由上至下第 i 列、由左至右第 j 行， 編號從 1 開始。他先選一格 (X, Y) 作為自己的養老用地， 剩下 N×N−1 格要平分給 N×N−13 個子孫。每個子孫剛好分到相連的 3 格，必須排成 L 型（不可為 1×3 直線）。

任兩個人不得分到同一格。請你設計分配方案，使得王老先生家中和平。

Input

本題採互動式評測。請實作以下函式（已於 lib0114.h 中宣告）： 

void solve(int N, int X, int Y);

其中 N 表示王老先生土地的邊長，(X, Y) 為他選的養老地座標。

對配分 30% 的測試資料，滿足 1 < N ≤ 8。

對配分 100% 的測試資料，滿足 1 < N ≤ 1024、1 ≤ X, Y ≤ N， 且存在正整數 K 使 N = 2K。

Output

請以 Report(x1, y1, x2, y2, x3, y3) 函式回傳答案。 每呼叫一次代表該子孫分到的三個格子為 (x1, y1)、 (x2, y2)、(x3, y3)。

三個格子之間的順序與多次 Report 之間的順序均不影響判定； 只要整體分配滿足規則即視為正確。若 Report 的輸入形狀錯誤、 土地重疊、或呼叫次數與題目要求不符，則判定為錯誤。

Sample

評分程式呼叫：

solve(4, 3, 4);

你的 solve 函式可依分治順序呼叫：

Report(2, 2,  2, 3,  3, 2);
Report(1, 1,  1, 2,  2, 1);
Report(1, 3,  1, 4,  2, 4);
Report(3, 1,  4, 1,  4, 2);
Report(3, 3,  4, 3,  4, 4);
